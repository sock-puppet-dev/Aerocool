Aerocool.ua — сайт бренда игровых и офисных кресел.

Это статический сайт, созданный с помощью генератора Hugo.
Сайт поддерживает два языка — русский и украинский, имеет SEO-оптимизацию и включает разделы 
с товарами, новостями, статьями, FAQ и контактами. 
Технологии, которые используются в проекте: Hugo, тема PaperMod, Tailwind CSS. 
Для деплоя и автоматизации используется платформа Netlify.

Структура проекта примерно такая:

    • Папка content содержит контент сайта.
    • Папка layouts хранит шаблоны Hugo и их переопределения.
    • Папка static содержит все статические файлы: изображения, стили и прочее.
    • Файл hugo.yaml — основной конфигурационный файл Hugo

Готовый сайт появится в папке public и его можно загрузить на хостинг Netlify.
Добавлять новый контент можно создавая markdown-файлы в папках content.
с нужными метаданными (title, description, keywords и прочее). 
Для удобства оформления используются специальные шаблоны — шорткоды,
например, для вывода последних статей или популярных тегов.
