Aerocool.ua — официальный сайт бренда игровых и офисных кресел.

Это статический сайт, созданный с помощью генератора Hugo. Используется тема PaperMod и стили Tailwind CSS. Сайт поддерживает два языка — русский и украинский, имеет SEO-оптимизацию и включает разделы с товарами, новостями, статьями, FAQ и контактами.

Технологии, которые используются в проекте: Hugo версии 0.148.2 с расширениями, тема PaperMod, Tailwind CSS версии 4.1. Для деплоя и автоматизации используется платформа Netlify.

Структура проекта примерно такая:

	•	Папка content содержит контент сайта, внутри есть папки для каждого языка — ru и uk.
	•	Папка layouts хранит шаблоны Hugo и их переопределения.
	•	Папка static содержит все статические файлы — изображения, стили и прочее.
	•	Файл config.yaml — основной конфигурационный файл Hugo, где задаются параметры сайта и мультиязычность.

Чтобы запустить сайт локально, нужно:

	1.	Клонировать репозиторий.
	2.	В командной строке перейти в папку проекта.
	3.	Запустить команду hugo server -D.
	4.	Открыть в браузере адрес http://localhost:1313, чтобы увидеть сайт.

Для сборки готовой версии сайта выполните команду hugo. Готовый сайт появится в папке public и его можно загрузить на хостинг (например, Netlify).

Добавлять новый контент можно создавая markdown-файлы в папках content/ru или content/uk с нужными метаданными (title, description, keywords и прочее). Для удобства оформления используются специальные шаблоны — шорткоды, например, для вывода последних статей или популярных тегов.
