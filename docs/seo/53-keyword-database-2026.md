## Как Пользоваться Базой

CSV-файл содержит строки вида:

| Поле | Значение |
| --- | --- |
| `id` | стабильный идентификатор строки базы |
| `lang` | язык ключа: `uk`, `ru`, `any` |
| `keyword` | ключевая фраза |
| `cluster` | мастер-кластер: бренд, категория, серия, товар, материал, механизм, сравнение, поддержка |
| `subcluster` | уточнение внутри кластера |
| `intent` | поисковый интент |
| `priority` | `P0`, `P1`, `P2`, `P3` |
| `target_url` | текущая или рекомендуемая целевая URL |
| `status` | `current`, `supporting`, `growth` |
| `notes` | пояснение, как использовать ключ без переспама |

Практическое правило: перед созданием новой страницы сначала фильтруйте CSV по `cluster`, `intent` и `target_url`. Если ключ уже имеет сильную целевую страницу, лучше усилить эту страницу, а не создавать дубль.

## Ключевые Зоны Роста

| Зона | Что Делать |
| --- | --- |
| Безбрендовый commercial | Усиливать существующие хабы и статьи, не создавать дубли под каждую форму запроса. |
| Home office | Расширить практические критерии: длительность дня, вентиляция, подлокотники, регулировка сиденья, сценарии квартиры/офиса. |
| Осанка и эргономика | Писать аккуратно: без медицинских обещаний, через настройку посадки, поддержку поясницы, подлокотники и движение. |
| Материалы | Добавить больше практических сравнений: тепло, уход, контакт, вентиляция, внешний вид, сценарии помещения. |
| Механики WING и XTAL | Усилить `dual backrest` и `fully replaceable design` видимыми объяснениями, FAQ и внутренними ссылками. |
| Транзакционные фразы | Использовать только там, где подтверждены цена, наличие, доставка, гарантия и условия покупки. |

## Правила Расширения

- Не создавать новую страницу, если текущая URL уже точно закрывает интент.
- Не смешивать на одной странице `купить`, `сравнить`, `что это` и `как выбрать`, если от этого расплывается основная задача.
- Украинские и русские ключи развивать параллельно, но не дословным переводом.
- Для `лучшее`, `краще`, `лучший` обязательно раскрывать критерии выбора.
- Для характеристик вроде `BIFMA`, `price`, `availability`, `warranty`, `delivery` использовать только подтвержденные данные.
- Не использовать `meta keywords` как основную SEO-стратегию.
- Не вставлять ключевые слова в текст механически. Ключ должен быть подтвержден смыслом блока, ссылками, характеристиками или FAQ.
