# Шаблон Страницы Серии

Обновлено: 2026-05-18.

Использовать для страниц серий в `content/products/<series>/_index.md` и `_index.ru.md`.

Как пользоваться новичку: страница серии описывает линейку целиком, а не один товар. Здесь нужно объяснить, чем серия отличается от других серий, какие варианты внутри нее есть и куда пользователю перейти дальше.

Целевой стандарт для проекта `Aerocool`: одна страница серии обычно пишется в формате `6000+` знаков основного текста на каждую языковую версию. Объем должен раскрывать позиционирование линейки, разницу моделей, сценарии выбора и переходы в конкретные товары.

## Поля Метаданных Страницы

```yaml
---
title: "Aerocool <SERIES> — серия эргономичных кресел"
h1: "Aerocool <SERIES>" # необязательно
description: "Серия Aerocool <SERIES>: ключевые особенности, модели внутри линейки, отличия по материалам и сценариям использования."
date: 2026-04-21T10:00:00+03:00
lastmod: 2026-04-21T10:00:00+03:00
summary: "Серия <SERIES> с понятным позиционированием, сравнением моделей и сценариями выбора."
slug: "<series>"
image: "cover.webp"
cover:
  image: "cover.webp"
  alt: "Серия Aerocool <SERIES> с эргономичными креслами"
  relative: true
  hiddenInSingle: true
schema_types: ["website", "collection", "organization", "breadcrumbs"]
---
```

## Редакционные Правила

- Видимый `H1` рендерит шаблонный слой; в теле страницы серии не добавлять markdown `# H1`.
- Если `title` получается длиннее, чем нужно для визуального заголовка, можно задать отдельный `h1`.
- В видимом тексте страницы серии не использовать обратные кавычки для inline-code. Точные технические обозначения, характеристики, SKU/MPN/GTIN, размеры, рейтинги и значения из таблиц выделять обычным жирным форматом: `**11D**`, `**SYNC5**`, `**75 мм**`. Широкие коммерческие SEO-фразы писать обычным текстом или обычным жирным выделением.
- Страница серии не должна быть просто списком карточек.
- Всегда объяснять, чем серия отличается от соседних серий.
- Серийная страница должна закрывать не только брендовый запрос серии, но и широкий средний интент: `игровые кресла`, `офисные кресла`, `компьютерные кресла` в пределах релевантной линейки.
- Если внутри серии есть `Racer / Loft Air / Mesh`, обязательно раскрывать разницу материалов.
- Обязательно вести в конкретные модели и в сравнения или гайды.
- Не приписывать всей серии отдельную функцию, если она есть только у одной модели внутри линейки.
